Python Scapy Networking

Developed a Python-based security tool to demonstrate the vulnerabilities of the Address Resolution Protocol (ARP) and evaluate network resilience against Man-in-the-Middle (MitM) attacks.

  • Adversary Simulation: Engineered a script using the Scapy library to inject spoofed ARP replies into a controlled network, successfully poisoning the ARP cache of a Windows 11 target to redirect traffic through a Kali Linux attacker machine

  • Defense Implementation: Evaluated and documented effective countermeasures, specifically the implementation of Static ARP entries to prevent dynamic cache manipulation and enhance host-level security
